# Security

## Supported versions

Only the latest commit on the default branch is actively considered for security fixes. There are no numbered Long-Term Support releases.

## Reporting a vulnerability

Please report security issues privately so we can address them before public disclosure.

- Prefer [GitHub Security Advisories](https://github.com/gertyhiler/lan-share/security/advisories/new) (adjust the owner/repo if you use a fork).
- If that is not available, open a **private** issue or contact the repository maintainers through a channel they publish on the repository homepage.

## Scope notes

lan-share is a small LAN HTTP tool for trusted networks. It is not hardened for exposure to the public internet. Do not bind it to untrusted interfaces or rely on it for sensitive data without additional controls.
